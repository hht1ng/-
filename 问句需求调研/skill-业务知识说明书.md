# 业务知识说明书 Skill

## 状态：非必选（文档提供时读取，未提供则跳过）

该部分是业务术语和领域知识的**参考来源**。业务知识中**可能包含指标、维度和过滤条件的信息**，读取后应在三类提取中交叉使用，而非仅用于"业务上下文理解"。

## 数据位置

数据位于用户上传的 `业务知识说明书` 独立文档中。

## 文档结构

| 列 | 用途 |
|----|------|
| 记录人 | 知识提供者/来源 |
| 术语名称 | 业务术语的标准名称 |
| 术语描述 | 详细的业务规则和知识内容 |
| 适用范围 | 该知识的适用场景或接口范围 |
| 备注 | 补充说明 |

## 读取策略

1. **检查文档是否提供**：检查用户是否上传了 `业务知识说明书` 文档
2. **文档已提供**：完整读取，将知识条目加载到上下文中
3. **文档未提供**：跳过

## 知识对三类提取的贡献

业务知识中的条目可能同时影响指标、维度、过滤条件的提取，读取后应按以下方式交叉使用：

### 对指标提取的帮助

| 知识类型 | 说明 | 用途 |
|---------|------|------|
| 术语默认含义 | 业务术语在不加修饰词时的默认指向，如"X"默认指X指标，明确加"Y"时才指Y指标 | 根据修饰词精确匹配指标，避免错误地将泛指词匹配到错误指标 |
| 指标同义词 | 业务中对同一指标的不同称呼，如"A指标又称B指标" | 补充指标维度说明书中未列出的同义词映射，扩大匹配范围 |
| 计算口径 | 某计算指标的公式定义，如"X比率 = 分子指标 / 分母指标" | 将复合概念拆解为原子指标的组合，确保分子分母都被输出 |
| 连带输出要求 | 问某类指标时需要默认连带输出的附属指标列表 | 自动补全问句未提及但业务上必须一起输出的指标 |

### 对维度提取的帮助

| 知识类型 | 说明 | 用途 |
|---------|------|------|
| 维度同义词 | 业务用语到标准维度属性的映射，如某业务简称=某维度属性 | 将非标准用语映射到指标维度说明书中的正式维度属性 |
| 组织层级关系 | 某组织术语包含哪些子组织的定义 | 将组织简称映射到具体的部门/BU维度值 |
| 术语归属 | 某个业务概念属于哪个维度类别 | 将模糊术语归类到正确的维度，如"XX"通常指某个客户/医院/产品维度 |
| 枚举值定义 | 某个分类维度下包含哪些具体取值 | 识别问句中出现的具体值属于哪个维度，反向匹配 |
| 范围定义 | 某个业务范围或定义市场包含的产品/客户清单 | 通过产品名反查其所属项目/市场范围，补全维度信息 |

### 对过滤条件提取的帮助

| 知识类型 | 说明 | 用途 |
|---------|------|------|
| 数据来源判定 | 问句中出现某关键词时应从哪个数据源取数 | 自动添加"数据来源=…"过滤条件 |
| 范围/定义市场 | 某个定义市场或业务范围包含的实体清单 | 问句出现"XX定义市场"时补全过滤条件中隐含的范围约束 |
| 关联键规则 | 跨数据源关联时应使用哪个字段作为关联键 | 涉及多数据源时提示正确的关联字段，避免用错关联键 |
| 计算标注规则 | 按数据来源应附加的标注说明 | 在过滤条件或备注中按规则自动添加对应的标注信息 |
| 特殊周期规则 | 特定项目/业务线的自定义时间周期定义 | 处理该业务线问句时使用自定义周期（如非自然年的财年）替代自然年 |

## 使用原则

1. **不做选择题**：业务知识提供的是补充信息，不与指标维度说明书形成优先级竞争，而是作为**附加匹配层**同时使用
2. **交叉检索**：同一条知识可能同时影响指标和过滤条件（如数据来源关键词既触发对应的外部数据指标，又添加数据来源过滤条件）
3. **结果合并**：业务知识匹配到的指标/维度/过滤条件，与从其他文档匹配到的结果合并，去重后输出
4. **不额外标注**：从业务知识推导出的指标/维度不标记为"需确认"，因为业务知识也是正式参考来源

## 读取流程

```
业务知识说明书（独立文档）
    │
    ├── 检查文档是否提供
    │       ├── 已提供 → 读取，提取三类映射：
    │       │         · 指标同义词/默认含义/计算口径
    │       │         · 维度同义词/组织层级/术语归属
    │       │         · 过滤条件规则/数据来源判定/标注规则
    │       └── 未提供 → 跳过
    │
    └── 继续后续处理（三类提取时均需参考本文档的知识）
```
