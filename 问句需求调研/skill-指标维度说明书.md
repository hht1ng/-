# 指标维度说明书 Skill

## 状态：非必选（文档提供时读取，未提供则跳过）

该部分是问句指标提取和维度匹配的**核心参考依据**，处理任何问句前若已提供则优先读取。

## 数据位置

数据位于用户上传的 `指标维度说明书` 独立文档中。

文档包含两部分内容：

| 内容 | 说明 |
|------|------|
| 指标定义 | 项目所有可用指标及其元数据 |
| 维度定义 | 项目所有可用维度及其属性 |

## 文档结构

### 指标定义

定义项目所有可用指标及其元数据。典型列包含：

| 列 | 用途 |
|----|------|
| 是否涉及标记 | 标记该指标在实际问句中是否常见（用于匹配权重调整） |
| 指标分类 | 指标所属大类，用于分类检索 |
| 指标名称 | 指标的标准名称，**匹配时的第一优先级** |
| 所属模块 | 该指标适用的业务模块/看板范围 |
| 说明 | 指标定义、同义词、计算口径、特殊注意事项 |
| 指标复杂度 | 指标的计算复杂度（原子/简单计算/复杂计算） |
| 指标类型 | 指标类型（原子指标/计算指标） |
| 数据源表 | 各模块对应的数据库表名 |
| 取值来源字段 | 数据库字段名 |
| 计算逻辑 | 计算指标的计算公式或说明 |
| 特殊逻辑 | 特定场景下的SQL处理规则 |

**使用方式**：
1. 全量读取指标行，构建 `指标名称 → 指标详情` 映射
2. 从 `说明` 列提取同义词映射
3. 匹配优先精确匹配指标名称，其次同义词模糊匹配
4. `是否涉及标记` 若为"待定"或类似标记，降低该指标的匹配权重

### 维度定义

定义项目所有可用维度及其属性。典型列包含：

| 列 | 用途 |
|----|------|
| 是否涉及标记 | 标记该维度在实际问句中是否常见 |
| 维度大类 | 维度所属大类，如：时间、地域、组织、产品 |
| 维度属性 | 维度的具体属性名称，**匹配时的第一优先级** |
| 维度成员 | 维度的具体取值示例，辅助识别问句中的维度值 |
| 说明 | 维度定义、特殊规则（如自定义周期定义） |
| 来源表 | 该维度对应的数据库表 |
| 来源字段 | 数据库字段名 |
| 计算逻辑 | 维度的计算/转换方式 |
| 备注 | 额外说明 |

**使用方式**：
1. 全量读取维度行，构建 `维度属性 → 维度详情` 映射
2. 从 `说明` 列理解维度特殊规则
3. 匹配时考虑维度属性的多种自然语言表达（同义词、缩写、全称）
4. `维度成员` 列可用于反向匹配：问句中出现某个成员值时反推所属维度

## 读取流程

```
指标维度说明书（独立文档）
    │
    ├── 读取指标定义 → 构建指标映射表
    └── 读取维度定义 → 构建维度映射表
```
